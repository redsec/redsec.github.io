redsec zone: firmware security research

We are researching firmware security for x86/x64, ARM and Power platforms including:

  • coreboot and SeaBIOS security
  • Firmware security analysis including virtualization and emulation
  • Firmware forensic analysis

redsec coreboot for QEMU

We are providing for research purposes our build of coreboot with SeaBIOS payload targeting (Windows) QEMU. To get latest build please visit our github repository or just pull the whole reposity using git: 

$ cd your_source_code_directory
$ git pull https://github.com/redsec/coreboot-reference

If you would like to debug it using gdb server (using IDA for example) you should run QEMU using qemu-win-coreboot-gdb.bat script from Windows folder of our repository. Otherwise use qemu-win-coreboot.bat.
